# AI Market Research Dashboard

Static investor/shareholder dashboard for AI operations layer market research.

## GitHub

The app is served from `index.html`, with supporting `styles.css` and `app.js`.

## Render

This repo includes `render.yaml` for a Render Static Site. In Render, create a new Blueprint, connect this repository, and deploy the `main` branch. Render will publish the repository root as the static site.
